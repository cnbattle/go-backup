go 写的一个简单的邮件备份工具。

# 使用
- cp config.example.toml config.toml
- 根据自己的需求修改配置

注意:
> 大文件/多文件/高频备份，请注意邮箱服务商附件相关的限制，如：QQ邮箱 单附件限制大小50MB，一天上限200MB。
